# Your React App Is Talking. Grafana Helps You Listen
Your React App Is Talking. Grafana Helps You Listen - Presented by Shlok Srivastava (Founder and Community Director, React Hyderabad)

## Description -

Your React app is constantly producing signals: slow routes, JavaScript errors, failed API calls, broken interactions, rage clicks, blank screens, and performance drops. The problem is that many frontend teams are not listening to those signals clearly.

In this talk, we will explore frontend observability from a React developer’s perspective. We will look at how Grafana and Grafana Faro can help capture browser-side telemetry such as errors, logs, performance data, and user-facing issues from production React applications.

We will also discuss how AI-assisted debugging can help developers make sense of these signals faster, identify patterns, and move from vague user complaints to actionable production insights.
